# OSIRIS.EXE — Canon Bible

The working wiki for the OSIRIS.EXE universe. Each file is a living, editable
slice of canon. The authoritative deep lore lives in the franchise bibles under
[`../docs/`](../docs/) (the `OSIRIS_EXE_Master_Franchise_Bible_v3_*` PDFs and the
*Split Archive*); these Markdown files are the quick-reference, version-controlled
layer on top of them.
